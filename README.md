# Task Master 🚀

A premium, modern React task management application built with Vite.

## ✨ Features

- **Glassmorphism UI**: Beautiful, translucent interface with backdrop filters.
- **Dynamic Gradients**: Modern and professional color palette.
- **Local Persistence**: Tasks are saved to your browser's LocalStorage.
- **Responsive Design**: Works perfectly on desktop and mobile.
- **Micro-animations**: Smooth entry animations for tasks.

## 🛠️ Tech Stack

- **React**: Core library for building the UI.
- **Vite**: Next-generation frontend tooling.
- **Vanilla CSS**: Custom styling for maximum performance and flexibility.

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/farizdzulhami/simple-react-task-master.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 📝 Project Structure

- `src/App.jsx`: Main application logic.
- `src/index.css`: Global styles and design system.
- `public/`: Static assets.
